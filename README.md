# Weather App

Weather App is a simple web application that provides real-time weather information for locations worldwide. It is built using HTML, CSS, and JavaScript and utilizes the OpenWeather API for weather data retrieval.

## Features

- **Real-Time Weather Data**: Get up-to-date weather information for any location.
- **Search Functionality**: Search for weather conditions in specific cities or regions.
- **User-Friendly Interface**: A clean and intuitive user interface for easy navigation.
- **Responsive Design**: The app is designed to work seamlessly on both desktop and mobile devices.
- **Detailed Weather Information**: View temperature, humidity, wind speed, and more.

## Getting Started

Follow these steps to get started with Weather App:

1. Clone this repository:
   ```bash
   git clone https://github.com/SubikshaSathiasai/Weather-App.git
   ```

2. Open `index.html` in your web browser.

3. Enter the name of the city or location you want to check the weather for, and click the "Search" button.
